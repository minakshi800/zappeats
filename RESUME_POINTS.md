## 📝 Short Description for Resume

**ZappEats** - A full-stack food delivery web application built with the MERN stack (MongoDB, Express.js, React, Node.js), featuring secure payment processing with Stripe, real-time order management, and a modern responsive UI with dark mode support. The platform includes user authentication, restaurant browsing, shopping cart functionality, and a comprehensive admin system with role-based access control.

## ⭐ Top 5 Most Important Resume Points

• **Developed a full-stack MERN food delivery platform** with React, Node.js, Express.js, and MongoDB, implementing RESTful APIs for restaurant browsing, menu management, order processing, and user authentication with JWT token-based security

• **Integrated Stripe payment gateway** with webhook handling for secure payment processing, implementing payment intent creation, customer checkout flow, and real-time payment status updates

• **Built responsive React frontend** using Vite, Tailwind CSS, and Framer Motion animations, implementing dark/light theme toggle with Context API for global state management across cart, authentication, and user preferences

• **Developed secure authentication system** using bcrypt for password hashing and JWT tokens with middleware for protected routes, implementing role-based access control (customer, restaurant owner, admin)

• **Architected RESTful API backend** with Express.js middleware for error handling, CORS configuration, request validation, and centralized error responses with proper HTTP status codes
